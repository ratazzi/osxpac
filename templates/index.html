OS X pac file generator

pac 文件地址示例：

http://{{ host }}/pac/127.0.0.1/7777/?v={{ time }}
http://{{ host }}/pac/127.0.0.1/7777/server_ip/?v={{ time }}

因为 Chrome 和 Firefox 只认 SOCKS5 而不认 SOCKS，Safari 和其他程序则相反，所以有了这个生成器，不同的是，此生成器根据域名所对应的 ip 地址来确定是否使用代理，作用类似于 chnroutes.
